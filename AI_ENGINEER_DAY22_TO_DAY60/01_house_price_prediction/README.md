# House Price Prediction

This project predicts median house values using a Random Forest regression model.

## Files

- `app.py` - Run predictions via command line.
- `model_training.py` - Train and save the model from the dataset.
- `dataset.csv` - Sample dataset for training.
- `requirements.txt` - Dependencies.
- `README.md` - Project documentation.
