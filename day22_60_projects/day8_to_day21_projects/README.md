# Day 8–21 Python Learning Pack

## Overview

This learning pack contains intermediate beginner projects covering OOP, file
handling, APIs, data parsing, NumPy, Pandas, Matplotlib and a mini sales analysis
project. Each project is a standalone, terminal-based Python script with comments
and input validation.

## Installation

Recommended libraries:

```
pip install numpy pandas matplotlib requests
```

## How to Run

Open a terminal, cd into this folder and run any script, for example:

```
python student_class.py
python csv_analysis.py
python sales_data_analysis.py
```

For weather API scripts, obtain an API key from OpenWeatherMap and set the
environment variable `OPENWEATHER_API_KEY` or enter it when prompted.

## Concepts Learned

- Object-oriented programming (classes and objects)
- File I/O and text handling
- Error handling and robust input
- HTTP APIs and JSON parsing
- NumPy array operations
- Pandas data analysis and cleaning
- Matplotlib visualizations and charts

## Learning Outcome

By working through these projects you will gain practical experience with data
manipulation, visualization, and API integration—useful skills for data analysis
and Python development.
