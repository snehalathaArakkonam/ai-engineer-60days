# Python Beginner Projects Pack

## Project Overview

This pack contains a set of small, beginner-friendly terminal Python projects designed
to teach core programming concepts through practical examples. Each project is
implemented as a standalone Python file and can be run independently.

## Folder Structure

python_beginner_projects/
│
├── calculator.py
├── age_calculator.py
├── temperature_converter.py
├── grade_calculator.py
├── atm_simulator.py
├── multiplication_tables.py
├── number_guessing_game.py
├── calculator_functions.py
├── student_management_system.py
├── contact_book.py
└── expense_tracker.py

## How to Run

1. Open a terminal and navigate to the `python_beginner_projects` folder.
2. Run any project using the Python interpreter, for example:

```
python calculator.py
```

Replace `calculator.py` with any other filename in the folder.

## Skills Learned

- Basic Python syntax and program structure
- Functions and modular design
- Input validation and error handling
- Control flow: loops and conditionals
- Data structures: lists and dictionaries
- Simple I/O and formatting
- Working with dates and random numbers

## Beginner Learning Path

1. Start with `calculator.py` and `calculator_functions.py` to learn arithmetic and functions.
2. Continue with `age_calculator.py` and `temperature_converter.py` for I/O and modules like `datetime`.
3. Try `multiplication_tables.py` and `number_guessing_game.py` to practice loops and randomness.
4. Move to `grade_calculator.py` and `atm_simulator.py` for multi-step input, validation and state.
5. Explore `student_management_system.py` and `contact_book.py` to practice lists and dictionaries.
6. Finish with `expense_tracker.py` which consolidates many concepts: lists, dictionaries,
   aggregation, and date handling.

## Notes

- All programs are intentionally simple and run in-memory (no database or file persistence)
  to keep them focused on learning core programming concepts.
- Each file includes detailed comments to help you understand how the code works.

Happy coding!
